\subsection{Detailed Description}
A container for encapsulating the particle list. 

This class contains a list of all particles and provides several functions for accessing them.

Amongst others, it provides functions for iterating over all particles and particle pairs and processing them using a certain function.

Function for iterating over unique particle pairs. 

This function uniquely iterates over all particle pairs and processes each by calling the provided function. While {\ttfamily iterate\-\_\-pairs} would process both the particle pairs (a,b) and (b,a), this function only processes (a,b).

This way the performance can be improved using Newton's third law. \hypertarget{classParticleContainer_adf9993d21144e346ad75239a16250b64}{\index{Particle\-Container@{Particle\-Container}!prepare\-\_\-forces@{prepare\-\_\-forces}}
